Real-time operating system for the ATmega328P on ArduinoUno, designed to facilitate real-time concurrency and make it easier for 
microcontroller programmers to implement pseudo-multitasking.

Features:

1- Developed using C++17.
2- Supports up to 6 tasks.
3- Manages CPU clock cycles.
4- Manages I/O operations.
